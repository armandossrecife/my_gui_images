# My GUI Images

Aplicação GUI para manipular imagens.

Instale as dependências:

```bash
pip install requests
pip install tqdm
pip install pillow
```

Execute o arquivo principal

```bash
python3 main.py
```

[Detalhes do programa](https://github.com/armandossrecife/my_gui_images/blob/main/detalhes.md)
